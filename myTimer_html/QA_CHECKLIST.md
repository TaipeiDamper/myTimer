# Aura Timer 驗證清單

## 建置檢查

- 執行 `npm run check`，確認 Vite build 成功且 `dist` 內產生 JS/CSS assets。
- 執行 `npm run preview`，確認預覽頁面按鈕、樣式與互動功能都正常。

## 核心流程

- 番茄鐘：套用最小、最大與異常輸入值，確認會被校正；跑完最後一組後應顯示「全部完成！🎉」。
- 導覽保護：在計時中、暫停中、已有碼表圈數或速寫歷史時切頁，應顯示確認視窗，不可靜默清空。
- 速寫練習：測試線上圖片、資料夾圖片、暫存圖片；線上圖片失敗時應改用 fallback，歷史紀錄應指向實際顯示的圖片。
- 碼表：開始、暫停、繼續、計圈與重設都應維持正確時間與圈數。

## UX 與無障礙

- 鍵盤：首頁卡片可用 Enter/Space 開啟；確認視窗可用 Tab 循環焦點並可用 Esc 關閉。
- 觸控：音量滑桿在手機或觸控裝置上應可直接操作。
- 小螢幕：在 320px 到 600px 寬度下，首頁卡片、控制按鈕、速寫工具列與 modal 不應水平溢出。
