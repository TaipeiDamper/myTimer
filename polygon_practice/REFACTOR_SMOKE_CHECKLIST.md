# Polygon Practice — 重構回歸檢查清單

每次 phase 完成後手動驗證（行為應與重構前一致）。

## 模式切換
- [ ] 單形練習 / 困難模式 / 自訂模式可切換
- [ ] 進行中切換模式會跳出確認並重設

## 單形 / 困難
- [ ] 「生成並開始」可產生形狀並進入計時
- [ ] 困難模式為聯集外框單一形狀
- [ ] 輪廓 / 填滿 / 反選三種顯示正確
- [ ] 反選（困難/自訂）為練習區內相反區域實心填滿

## 計時流程
- [ ] 計時倒數顯示正確
- [ ] 暫停 / 繼續（按鈕與空白鍵）
- [ ] 時間到自動揭露答案
- [ ] 「揭露答案」顯示基準點
- [ ] 「重新生成」/「重新選點」可用

## 自訂模式
- [ ] 點格選頂點、點最後一點撤銷、點①或「完成選點」閉合
- [ ] 閉合後自動儲存並可繼續畫下一個
- [ ] 「開始練習」需至少一個已儲存多邊形
- [ ] 「清除上一個」：有選點時清選點，否則刪最後一個已存形狀
- [ ] 「清除全部」清空所有
- [ ] 編輯時可切換顯示模式預覽

## 格子與 UI
- [ ] 觀察格子 / 生成格子開關
- [ ] 自訂模式有選點或已存形狀時格子設定鎖定
